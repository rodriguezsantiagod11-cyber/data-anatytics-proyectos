
## Goal
Explore business data through an interactive dashboard to identify patterns and trends in sales performance.

## Dataset
Business dataset used to analyze sales performance and category distribution through interactive visualizations.

## Analysis Process

- Connected the dataset to Tableau.
- Created visualizations to explore sales distribution across categories.
- Built an interactive dashboard with filters to explore the data.
- 
## Key Insights
- Sales performance varies across product categories.
- Some categories generate higher revenue compared to others.
- Interactive filters allow deeper exploration of the data.
