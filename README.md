# Data Analytics Portfolio

This repository contains a collection of data analytics projects developed using different tools and technologies such as Excel, Tableau, Power BI, and Python.

The objective of this portfolio is to demonstrate skills in data analysis, data visualization, and problem-solving using real datasets.

## Tools Used

- Excel (Data cleaning, dashboards, analysis)
- Tableau (Interactive dashboards and data visualization)
- Power BI (Business intelligence dashboards)
- Python (Data analysis using Jupyter Notebook)

## Projects

### 1. Excel Sales Analysis
Analysis and visualization of sales data using Excel dashboards, pivot tables, and data cleaning techniques.

### 2. Tableau Dashboard
Interactive dashboard built in Tableau to explore business data and generate insights through visual analytics.

### 3. Power BI Business Analysis
Business intelligence dashboard created with Power BI to analyze performance indicators and key metrics.

### 4. Python Data Analytics
Exploratory data analysis performed in Python using Jupyter Notebook to extract insights from a dataset.
